# ADR-0002: AI-Assisted Development Workflow

## Status
Accepted

## Context
The project was developed using AI-assisted software construction techniques.

## Decision
Use AI tools for:
- code generation
- testing support
- documentation
- code review

Human developers remain responsible for:
- architecture decisions
- validation
- integration
- security review

## Consequences

### Positive
- Faster implementation
- Improved productivity
- Better documentation

### Negative
- Risk of hallucinated code
- Need for careful review
- Possible overengineering
